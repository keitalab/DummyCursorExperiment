# Dummy Cursor Program Unity ver.

ダミーカーソルをUnity で実装．

## 機能

- [x] 遅延
- [x] C/D 比
- [ ] 発見するまでの経過時間
- [ ] 他人のカーソル再生
- [ ] リアルタイム変更
- [ ] 実験モード
- [x] デバッガモード

## Requirements for development

 - git
 - Unity v2019.1.4f1

## Development

主に `/Assets/Script/` 以下を編集する．だいたいのスクリプトはSceneに配置しているGameObjectにアタッチして使う．

`/Assets/Scripts/Public` の中に全体で共有する変数や関数が入れてある．
